## Findings + statut (✅ corrigé ce jour / ⏳ en attente de données)

### ✅ Corrigés (quick wins, commits `9fae46a`/`3ce4e61`/`39b39af`)

**1. Pages `noindex` dans le sitemap** · Impact Moyen
- *Était* : `merci` et `thank-you` présents dans le sitemap tout en portant `noindex` (signal contradictoire).
- *Fix* : `sitemap({ filter: (page) => !/\/(merci|thank-you)\/?$/.test(page) })`. Vérifié absent du sitemap.

**3. Meta descriptions hors plage** · Impact Moyen
- *Était* : pages services/index/à-propos **tronquées (>160)** (jusqu'à 197) ; **20 pages tags quasi-vides (12-22 car.)** (« Articles : seo »).
- *Fix* : `metaDescription` dédiée (~150 car.) par service + raccourci des clés `services/about.page.metaDescription` ≤160 ; gabarit `blog.tagMetaTemplate` → descriptions de tags 75-96 car. Vérifié : 0 description >160.

**4. Titres d'articles trop longs (>60)** · Impact Faible-Moyen
- *Était* : `<title>` d'articles jusqu'à 96-98 car. (tronqués en SERP).
- *Fix* : champ `seoTitle` optionnel (frontmatter) ; `BlogLayout` l'utilise pour `<title>` (suffixe marque ajouté seulement si ≤60) ; `seoTitle` ≤54 car. ajouté à 13 articles. `<h1>` d'affichage inchangé. Vérifié : 0 `<title>` d'article >60.

**2a. `openingHours` du schema local** · Impact Haut (partiel)
- *Fix* : `OPENING_HOURS = ['Mo-Fr 08:00-18:00','Sa 09:00-13:00']` ajouté au nœud `Organization`. Vérifié présent.

### ⏳ En attente (dépendent de vraies données / d'un asset)

**2b. NAP / schema local incomplet** · Impact Haut · Prio Haute
- `address` = région + pays seulement (pas de rue/ville/CP) ; `telephone` = placeholder ; **`sameAs` absent** (réseaux sociaux en `#` → invalides tant qu'il n'y a pas d'URLs réelles).
- *À faire quand dispo* : adresse réelle complète ; `sameAs` avec les vraies URLs sociales.

**5. Aucune image OpenGraph** · Impact Moyen · Prio Moyenne
- `og:image` absent sur les 64 pages (génération Satori différée).
- *À faire* : image OG statique par défaut, ou génération Satori (par page/article).

- **Meta descriptions d'articles** un peu courtes (101-138) — OK, pourraient viser ~150.
